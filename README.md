# FOOD-MATIC

> Browser-based food development tool — recipe management, ingredient costing, yield tracking.

**Live demo:** deploy to GitHub Pages (see INIT.md)

---

## Features

- **Ingredients** — track purchase qty, price, auto-calculate cost per unit
- **Recipes** — build menus with ingredient lists, yield, servings
- **Cost Analysis** — food cost %, gross profit, cost per serving
- **Dashboard** — quick overview of all data
- **JSON Import/Export** — portable data, no lock-in
- **Works offline** — all data in `localStorage`
- **Mobile-friendly** — responsive dark UI

## Quick Start

```bash
# Clone and open
git clone https://github.com/YOUR_USERNAME/FOOD-MATIC.git
cd FOOD-MATIC
open index.html   # or double-click
```

## Deploy to GitHub Pages

Settings → Pages → Source: main branch / root → Save
